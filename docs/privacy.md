# 隐私说明

OfferAgent 是开源个人工具，不提供账号体系、团队云端空间或项目自有服务端报告库。当前 Web 版会为了一致性和反馈恢复，在用户本机浏览器 `localStorage` 中保存少量运行数据。

- 不需要注册或登录。
- 不上传简历、JD、报告或反馈到 OfferAgent 自有服务器。
- API Key 只在当前页面内存中临时使用，不写入 `localStorage`。
- 基础报告缓存会按输入指纹保存在本机浏览器，用于复用相同输入的运行结果。
- 语言 artifact 会随基础报告缓存保存，用于中英文展示复用。
- 面试官人工反馈历史会按输入指纹独立保存在本机浏览器，用于刷新或命中缓存后恢复最近反馈。
- 基础报告缓存不包含 API Key，也不把人工反馈写入基础报告缓存。
- 页面提供“清理本机缓存”入口，可清除运行缓存和反馈历史。
- 报告仍需要用户自行下载保存。

如果用户选择在浏览器中直连大模型服务商，输入内容会发送给对应服务商。请自行确认服务商的隐私政策和数据处理规则。

- 公共设备或不可信环境中，不建议输入真实敏感简历、JD 或 API Key。
- 如果已经在公共设备输入过真实信息，请使用页面“清理本机缓存”，并清理浏览器站点数据。
- 目标岗位选择、EvidenceGraph、OfferSimulationRun 和 FeedbackDistillation 会在当前页面运行状态中生成；基础报告 artifact 和反馈历史可能按输入指纹写入本机缓存。
- 当前 Web 版提供 PDF 下载，不提供面向用户的评测 JSON 导出。
